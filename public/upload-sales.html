제품별 매출량 업로드

ERP 거래명세서 파일을 업로드하면 자동으로 필터링 후 제품별 매출량이 저장됩니다.

자동 적용 필터 규칙

아래 데이터는 자동으로 제외됩니다
일반사업소 = '하자조치내수'
브랜드구분 = '알로소'
제품구분에 '소파', '철제' 포함 또는 '외주상품'
매출수량이 0 이하 (마이너스)
✅ 국가 구분: 제품구분 컬럼의 '베트남상품' → 베트남 / 그 외 → 국내
✅ 기간 구분: 매출일자에서 년-월(YYYY-MM) 자동 추출
✅ 제품 매핑: 단품코드 → 제품구분(item) 자동 변환 (맵 미등록 코드는 제품구분 값 그대로 저장, 동일 item 합산)
✅ 브랜드: 브랜드구분 컬럼 값(시디즈/퍼시스/일룸/데스커 등)을 그대로 저장 — KPI는 시디즈만, 제품별 클레임은 전 브랜드 합산
✅ 중복 제거: 수주번호+라인번호 기준으로 파일 간 중복 행 자동 제거
✅ 다중 파일: 여러 파일을 동시에 선택하면 합산 처리됩니다
✅ 재업로드: 동일 제품+년월+국가가 이미 있으면 덮어씁니다
